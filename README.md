# Streamlit stock App
## Stock information app using Streamlit 

- Streamlit
- yfinance

#### Get US Stock information using python yfinance
#### Show stock information and Historical data
### Run Application
#### Install Requirements
```python
pip install -r requirements.txt
```
#### Run Application
```python
streamlit run stock.py
```
